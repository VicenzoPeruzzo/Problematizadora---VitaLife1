VitaLife - Sistema de Gestão de Saúde
Este repositório contém o código-fonte do projeto VitaLife, desenvolvido como parte da disciplina "Problematizadora". O sistema integra médicos, pacientes e unidades de saúde em uma plataforma unificada.

Sobre o Projeto
O VitaLife é uma aplicação web para organização de serviços de saúde. A arquitetura é modular, separando backend (API), modelos de dados (Entities) e frontend (Pages).

Principais Funcionalidades
Gestão de Médicos: Cadastro e visualização de profissionais.

Agendamento: Criação e gerenciamento de consultas.

Unidades de Saúde: Localização e gestão de clínicas/hospitais.

Lembretes: Notificações para consultas e medicamentos.

Mapa: Visualização geográfica das unidades.

Assinaturas: Gestão de planos recorrentes.

Tecnologias
Linguagem: JavaScript (Node.js)

Infraestrutura: Docker

Estrutura de Pastas
API: Rotas e lógica do backend.

entities: Modelos de dados.

pages: Interface do usuário (Frontend).

server.js: Arquivo principal do servidor.

Dockerfile: Configuração de container.
